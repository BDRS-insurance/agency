# BDRS Associates Insurance Agency
> **Building Dependable Risk Solutions.**

BDRS Associates is a professional, responsive web application for a Philippine-based insurance agency. It provides a streamlined interface for clients to explore protection plans and request professional insurance quotes.

---

## 🚀 Key Features

* **Modern Dark UI:** High-contrast design built with Tailwind CSS.
* **Accredited Partners Grid:** High-visibility display of partner insurance companies including Pacific Cross, Paramount, PhilBritish, Asia United, Bethel, and Maagap.
* **Interactive History:** Clickable partner cards that reveal corporate legacy and expertise.
* **Product Inquiry System:** Dedicated sections for Life, HMO, and Non-Life insurance with detailed benefit modals.
* **Automated Quote Request:** Functional contact form integrated with **Web3Forms**.

---

## 🛠️ Tech Stack

* **React 18** - Frontend Framework
* **TypeScript** - Type Safety
* **Tailwind CSS** - Styling & Layout
* **Lucide React** - Iconography
